Everything you need for SVG in Python

From parsing to pixel-perfect rendering, svglab covers the entire SVG workflow.

code

Parse & Serialize

Read SVGs from strings, files, or byte streams. Write them back with full control over formatting — or just call .save().

account_tree

Full Element Model

All 80 elements and 295 attributes from the SVG 1.1 spec, each with a dedicated Python class and typed properties.

transform

Transforms & Path Data

First-class support for affine transforms, path commands, and "reification" — baking transforms directly into geometry.

speed

Optimize & Minify

Reduce file size by lowering precision, converting coordinates, shortening color values, and removing redundant data.

verified_user

Typed & Validated

Every attribute is a real Python type. Pydantic validates at runtime; your IDE catches the rest at edit time.

image

Render & Analyze

Rasterize to PNG via resvg, compute bounding boxes, and extract pixel masks — all from Python.

See it in action

Parse an SVG, tweak it, and save — in four lines. 

from svglab import parse_svg, Color, Formatter

svg = parse_svg("drawing.svg")
svg.find("rect").fill = Color("coral")
svg.save("output.svg", formatter=Formatter.optimized())
